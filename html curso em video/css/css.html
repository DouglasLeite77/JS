Algumas tags

text-align - aliamento da linha

height - altura do texto

width - largura do texto

border - borda

padding - preenchimento da borda interno

margin - preenchimento da borda externo

outline - contorno

display: nome - esconde o texto

display: block - mostra o texto escondido

Pseudo-classes

div:hover

hover - mudanças quando o mouse estiver em cima

active - mudanças ao clickar

visited - mudanças depois que algo foi clickado

Pseudo-elementos

usamos :: para Pseudo-elementos

::before - adicionar algo antes do texto

::after adicionar algo depois do texto

Filhos

div > p - Para fazer mudanças no <p> que estiver dentro da div

Tipos de caixa

box-level - quebra a linha, preenche uma linha inteira e pula para proxima para continuar o conteudo
