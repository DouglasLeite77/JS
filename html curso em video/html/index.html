Aqui temos a tag para quebrar uma linha <br>

Aqui temos as tags para adicionar alguns simbolos especiais: ® © ™ € £ ¥ ¢ $ ℜ Δ

Formatações

Para colocar em negrito usamos o <strong>

Para colocar em itálico usamos o <em>

Fazer um marcação de uma parte do texto usamos <mark>

Texto grande e texto pequeno: <big> e <small>

Para marcar um texto como excluido usamos <del>

Para marcar um texto inserido usamos <ins>

Para sobrescrever usamos <sup> exemplo: x20+3

Para subescrever usamos <sub> Exemplos: H2O

Podemos utilizar o <pre> para aparecer todos os espaçamentos. Exemplos:

        
    num = int(input("digite um número"))

    print('Fim do programa')

Utilizamos também o <code> para formatar melhor o espaço entre as letras quando formos apresentar um código

Citações simples usamos <q>

Citações complestas usamos o <blockquote>

Para abreviações usamos <abbr>

Listas Ordenadas

  1. Ultilizamos o <ol> para criar listas ordenadas e dentro delas utilizamos o <li> para escrever cada item da lista
  2. Dentro do <li> podemos escolher o type e o start

Listas não ordenadas

  • Utilizamos o <ul> para criar listas não ordenadas e utilizamos também o <li> para escrever cada item da lista
  • No <ul> so temos 3 tipos de type que são: disc, circle e square

Lista de definição

Usamos <dl> para criar uma lista de definição

Dentro da lista utilizamos <dt> para o termo e <dd> para a definição

HTML
Linguagem de marcação para a criação do conteúdo de um site

Links

Para inserir um link clicavel em um texto do site usamos <a>

Sempre impostante colocar as tags "target=_blank" e "rel=external" que fazem com que o link seja aberto em outra aba

Para links internos podemos deixar sem tags ou colocar "rel=self"

./ refere a uma próxima pasta e ../ refere a uma pasta anterior

Para disponibilizar para download vamos colocar as tags "download=(o nome do arquivo) e "type=application/pdf" por exemplo, para ver qual é o type de cada arquivos temos esse site

Imagens e os diferentes tamanhos

Primeiro vamos ter que produzir a imagens em tamanhos diferentes, por exemplo 1000px, 700px e 300px.
Em seguida abrimos a tag <picture> e adicionamos ou a maior foto ou a menor com a tag <img>.
Adicionado a imagem, abrimos uma tag <source:media:type> (sempre acima da tag da imagem) com "media=(max)" e os px com uma margem de mais 50px

Áudios

A maneira mais fácil de implementar um áudio no site é com o <audio>
